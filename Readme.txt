Pentru a testa cronjob-ul in terminal accesam:
<calea proiect> php cron.php job
Data de start si data de sfirsit se introduc din terminal, iar denumirea orashului se skimba in fisierul project/protected/commands/JobCommand.php
Datele care se repeta nu se mai introduc in baza.
La sfirsit se afiseaza cite inregistrari au fost introduse in baza de date.
